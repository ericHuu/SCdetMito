SCdetMito Detailed Tutorial

This tutorial outlines an internet-independent SCdetMito workflow using the bundled multi-donor PBMC demo.

Installation

Install SCdetMito from GitHub with:

install.packages("remotes")
remotes::install_github("ericHuu/SCdetMito@v1.4.1", build_vignettes = FALSE)

Then load the package:

library(SCdetMito)

Load Demo Data

pbmc <- load_demo_pbmc()
table(pbmc$sample)
table(pbmc$group)
summary(pbmc$mitoRatio)

The demo is derived from public 10x Genomics GEM-X Human PBMC Donor 1-4 datasets and downsampled to 2,500 cells per donor. It is intended for software demonstration and testing only.

Ensure Mitochondrial Fraction Metadata

pbmc <- ensure_mito_ratio(
  pbmc,
  mito_col = "mitoRatio",
  species = "human"
)

If automatic feature detection is not appropriate, provide a complete mitochondrial feature vector retained in the object:

mito_features <- grep("^MT-", rownames(pbmc), value = TRUE)
pbmc <- ensure_mito_ratio(
  pbmc,
  mito_col = "mitoRatio",
  species = "human",
  mito_features = mito_features,
  overwrite = TRUE
)

Run SCdetMito

det <- SCdetMito(
  pbmc,
  sample_col = "sample",
  mito_col = "mitoRatio",
  species = "human",
  tissue = "PBMC",
  sample_cutoff_method = "reference_guided",
  return_details = TRUE,
  write_plots = FALSE,
  write_tables = FALSE
)

det$sample_cutoff_summary

Apply Groupwise QC

qc <- SCQCmulti(
  pbmc,
  sample_col = "sample",
  group_col = "group",
  mito_col = "mitoRatio",
  cutoff_strategy = "groupwise",
  scdet_options = list(sample_cutoff_method = "reference_guided"),
  species = "human",
  tissue = "PBMC",
  min_genes = 0,
  min_counts = 0,
  removeDouble = FALSE,
  write_plots = FALSE,
  write_tables = FALSE
)

qc@misc$SCdetMito_QC$group_plan

Compare Strategies

bench <- SCQCbenchmark(
  pbmc,
  sample_col = "sample",
  group_col = "group",
  species = "human",
  tissue = "PBMC",
  min_genes = 0,
  min_counts = 0,
  removeDouble = FALSE,
  run_downstream = FALSE,
  plot = FALSE,
  output_dir = tempdir()
)

bench$recommended_strategies

The recommended strategy is ranked under the selected scoring profile. It is not a universal QC optimum.
