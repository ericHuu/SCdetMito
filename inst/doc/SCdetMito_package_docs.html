Purpose

SCdetMito is a lightweight R package for retention-loss enrichment-based adaptive mitochondrial ratio cutoff recommendation in single-cell RNA-seq quality control. The package is designed for one-sample, multi-sample, and group-aware integrated Seurat workflows.

The package should be used as a quality-control recommendation and auditing tool. It does not claim that a single universal mitochondrial cutoff exists for all tissues, species, chemistries, or disease contexts, and it should not be framed as a classical breakpoint model.

Example Data

The installed package includes lightweight objects for examples and checks:

  • load_demo_seurat(): synthetic multi-group toy data for fast examples.
  • load_pbmc3k_demo(): lightweight public PBMC3K subset with 300 cells and 700 features.

The bundled objects are intended for package examples and checks, not for large-scale validation claims.

For a real public dataset, users can optionally call load_pbmc3k_online(method = "10x"), load_heart10k_online(), or load_ifnb_online(). PBMC3K is a small public smoke-test example, heart10k is a larger real-tissue example, and IFNB is a multi-condition workflow example. Online datasets are downloaded or loaded only when the corresponding function is called. Internet access is not required for standard package checks.

Core Workflow

library(SCdetMito)

pbmc3k <- load_pbmc3k_demo()

det <- SCdetMito(
  pbmc3k,
  sample_col = "sample",
  mito_col = "mitoRatio",
  min_cut = 0.01,
  max_cut = 0.25,
  return_details = TRUE,
  write_plots = FALSE,
  write_tables = FALSE
)

det$cutoff
det$sample_cutoff_summary
det$retained_cell_profile
det$interval_loss_table

Strategy Guide

  • consensus: use one all-sample sample-supported global cutoff.
  • strictest: use the most conservative sample-level cutoff.
  • groupwise: use group-specific cutoffs when each biological group contains multiple samples.

Available detection tests can be listed with:

SCdetMito_methods()

Primary default:

  • loss_test = "mad_zscore"
  • p_adj_method = "BH"
  • sample_cutoff_method = "largest_drop"

Useful sensitivity checks:

  • loss_test = "empirical_tail"
  • loss_test = "poisson_tail"
  • p_adj_method = "none" for unadjusted exploratory review
  • p_adj_method = "holm"
  • SCdetMito_sensitivity() for parameter stability checks
